Système de transcription du langage des signes

C'est une plateforme de transcription de langage de signe dévéloppé avec le framework web Django qui utillise le modèle Deep Learning pré-entrainé (SSD MobileNet).

SSD MobileNet une architecture de réseau de neurones convolutifs (CNN), elle est choisie pour sa combinaison de précision et d'efficacité. Cette architecture est spécialement conçue pour des tâches de détection d'objets en temps réel, ce qui en fait un choix idéal pour la détection des signes dans des séquences vidéo.

Site web de vente en ligne

Le projet que nous avons développé est un site web de marketplace basé sur le framework Django. Une marketplace en ligne est une plateforme qui permet aux vendeurs de proposer leurs produits à la vente et aux acheteurs de les parcourir, de les sélectionner et de les acheter, tout cela en un seul lieu centralisé.

L'objectif de ce projet est de créer une solution web efficace et évolutive pour faciliter cette activité commerciale. Votre site web a été conçu pour offrir une interface facile à utiliser tout en restant assez flexible pour permettre une croissance à long terme afin de faciliter la mise en relation entre vendeurs et acheteurs en ligne.
